# Aura Estética – Clínica Dermatológica (One Page)

Projeto institucional **one-page** de uma clínica dermatológica e estética premium, desenvolvido com foco em **design elegante**, **experiência do usuário**, **responsividade** e **conversão**.

Este projeto foi criado como **modelo de portfólio profissional**, podendo ser facilmente adaptado para clínicas, consultórios e negócios da área da saúde e estética.

---

## 🖥️ Visão Geral

O site apresenta uma navegação fluida, layout clean e linguagem visual premium, transmitindo confiança, sofisticação e profissionalismo.

Principais objetivos:
- Apresentação institucional
- Destaque dos tratamentos
- Geração de leads via agendamento
- Facilidade de contato

---
