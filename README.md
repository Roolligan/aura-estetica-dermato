# Aura Estetica

Site one-page de portfolio para a clinica dermatologica Aura Estetica.

## Stack
- Vite
- React
- TypeScript
- CSS puro

## Como rodar
```
npm install
npm run dev
```

## Estrutura de pastas
```
src/
  app/
  components/
  data/
  hooks/
  styles/
```

## Checklist de features
- Scrollspy no menu
- Modal de agendamento
- Filtro por chips em tratamentos
- Carrossel simples de depoimentos
- Botao flutuante de WhatsApp no mobile

## Notas de customizacao
- Edite cores em `src/styles/tokens.css`
- Atualize textos em `src/data/mock.ts`
- Ajuste tratamentos e precos em `src/data/mock.ts`
